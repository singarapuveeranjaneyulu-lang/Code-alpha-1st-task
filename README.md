# CodeAlpha Simple E-Commerce Store

## Project Overview

This project was developed as part of the CodeAlpha Full Stack Development Internship.

A simple E-Commerce web application where users can:

- Register and Login
- View Products
- Add Products to Cart
- Place Orders
- Store Users, Products and Orders in MongoDB

---

## Tech Stack

### Frontend

- HTML
- CSS
- JavaScript

### Backend

- Node.js
- Express.js

### Database

- MongoDB

---

## Features

### User Authentication

- User Registration
- User Login
- JWT Authentication

### Product Management

- Product Listing
- Product Details

### Shopping Cart

- Add to Cart
- View Cart
- Total Price Calculation

### Order Processing

- Place Order
- Store Orders in Database

## Installation

### Clone Repository

git clone <repository-url>

### Install Dependencies

cd backend

npm install

### Run Server

npm start

### Open Frontend

Run index.html using Live Server.
